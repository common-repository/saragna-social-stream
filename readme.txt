== Description ==

The Social Stream plugin lets you retrieve posts/videos/images from different social networks in a grid format from the newest to the oldest.


== Installation ==

**This section describes how to install the plugin and get it working**


= Automatic installation =

To do an automatic install of saragna-social-stream, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "saragna-social-stream" and click Search Plugins. Once you have found it you can install it by simply clicking "Install Now".

= Manual installation =

**Uploading via WordPress Dashboard**

1. Download `saragna-social-stream.zip`
2. Navigate to the 'Add New' in the plugins dashboard
3. Navigate to the 'Upload' area
4. Select `saragna-social-stream.zip` from your computer
5. Click 'Install Now'
6. Activate the plugin in the Plugin dashboard

**Using FTP**

1. Download `saragna-social-stream.zip`
2. Extract the `saragna-social-stream` directory to your computer
3. Upload the `saragna-social-stream` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard


The WordPress codex contains [instructions on how to install a WordPress plugin](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
